Étape 1 : Les Fondations du Design (UI/UX)
Avant de coder, vous devez apprendre à structurer visuellement votre idée. [1]
Le Wireframing : Apprendre à dessiner des squelettes en noir et blanc de vos pages.
L'Architecture de l'information : Comprendre comment organiser le contenu logiquement.
Le Design Système basique : Maîtriser le choix d'une palette de couleurs et de deux polices maximum
## Étape 2 : Le Visuel de l'Application (Frontend)

C'est ce que l'utilisateur voit et utilise sur son écran.

### Concepts à maîtriser

- **HTML5** : Structure d'une page web (titres, paragraphes, formulaires, boutons).
- **CSS3** : Apparence de la page (couleurs, espacements, typographie, Flexbox, Grid).
- **Responsive Design** : Adaptation de l'interface aux mobiles, tablettes et ordinateurs.

---

## Étape 3 : Le Cerveau de l'Application (Logique Frontend)

Permet de rendre l'application dynamique et interactive.

### Concepts à maîtriser

- **Variables et Types de données** : Stocker des informations temporaires (nom, email, panier, etc.).
- **Fonctions** : Regrouper des actions réutilisables (calculer un prix, valider un formulaire, etc.).
- **DOM (Document Object Model)** : Modifier le contenu HTML et CSS en temps réel selon les actions de l'utilisateur.

---

## Étape 4 : Le Stockage des Données (Backend & Base de Données)

Permet de conserver les informations de manière permanente.

### Concepts à maîtriser

- **Modèle Conceptuel de Données (MCD)** : Définir les entités et relations (Utilisateurs, Produits, Commandes, etc.).
- **CRUD** :
  - Create
  - Read
  - Update
  - Delete
- **Requêtes HTTP / API** : Communication entre le frontend et le backend pour récupérer ou enregistrer des données.

---

## Étape 5 : La Mise en Ligne (Déploiement)

Permet de rendre l'application accessible sur Internet.

### Concepts à maîtriser

- **Routage & Domaine** : Associer un nom de domaine (ex : `monapp.com`) à votre application.
- **Variables d'environnement** : Stocker les informations sensibles (clés API, mots de passe, secrets).
- **Hébergement** : Déployer l'application sur un serveur ou une plateforme cloud.
